Comparing Spatial plaquette and space-time plaquette values of MILC vs SIMULATeQCD calculations

Build MILC:

  1. Download it from milc_download.
  2. Goto hvy_qpot , copy Makefile from main directory using cd hvy_qpot/ && cp ../Makefile .
  3. Edit the Makefile (choose double precison and comment out WANTQIO = true using WANTQIO= # true)
  4. Make the executable su3_hqp_hyp using make su3_hqp_hyp
  5. Remember that the parameter files are in /test/ directory of each applications of MILC. This directory also contains the sample input and sample output, checked them recently and they agrees upon over all observables.
  6. The binary files are in milc_qcd-7.8.1/binary_samples
  7. Note that the each executable applications has control.c file which contains the main code of applications.
