The milc vs SIMULATeQCD wilson loop Computation

MILC code calculates temporal wilson loops.

The plaquette value of b=6.354 or any other beta matches with that of SIMULATeQCD code. But the discrepency starts to appear as we look for values of larger wilson loops. It starts right from disagreement in smallest rectangular wilson loop.

By looking at the "control.c" program of MILC/hvy_qpt. we can comment out the coulomb gauge fixing and axial gauge fixing and see if the answers try to match the original configuration or not.

If we don't use coulomb and axial gauge fixing on milc

If we don't use the coulomb gauge fixing and axial gauge fixing on milc, then the results are same if we use coulomb gauge fixing on milc or not. i.e. for a given gauge configuration the results are same wheather we fix to the coulomb gauge or not. the results change for the given configuration if we use the axial gauge fixing.
