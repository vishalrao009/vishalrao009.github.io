Largest array storage and modification in CPU and GPU

In this article, we shall discuss how one can utilize CUDA parallel programming in modifying the largest array possible on any computer.

We will follow the following steps:

  1. First, find the largest possible array of integers can be stored in any given computer.
  2. Then we will modify the largest possible array using serial programming (C++ standard).
  3. We, shall then write a program in CUDA C++ (which is subset of C++ standard) and will implement parallel programming to the above task.

Finding Largest array of integer

Basically, we are looking for largest size of array of integers. Any signed/unsigned integer take 4 bytes. To calculate the maximum size of array we need to know the amount of available memory.
Here comes the interesting part, there are two types of memory program can use,

  1. Stack memory
  2. Heap Memory

Stack memory and heap memory are two types of memory used by computer programs for different purposes. They differ in terms of allocation, deallocation, and scope. For more, find Memory allocation

Stack holds limited amount of memory. It varies from computer to computer depending upon OS and compiler from 1MB to 8MB.

Therefore it is better to allocate the memory on run time(dynamic memory allocation), rather than compile time(such memory allocation is automatically handled by stack memory)
